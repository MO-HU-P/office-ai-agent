## スライドデザインガイド（PPTXの依頼のとき、システムが自動で添付する指針）

ユーザーがデザインを指定しないときは、以下に従って「おまかせで気の利いたデザイン」に仕上げる。
ユーザーが色・レイアウト・枚数などを指定したときは、常にユーザーの指定を優先する。

### 前提（テンプレートに任せること）
- ppt_create が作るファイルは16:9(幅33.87cm x 高さ19.05cm)のデザイン済みテンプレート。
  背景・フォント・タイトルの見た目・表のヘッダー色・図形とグラフの色はテンプレートが自動で整える。
  したがって**色やフォントを自分で細かく指定しない**（指定するほど統一感が崩れる）。
- 使う色を明示したいときは次のパレットから選ぶ:
  青 #1A73E8（基本・強調） / 緑 #188038（良い・増加） / 赤 #D93025（悪い・注意） /
  黄 #F9AB00（中間・注目） / 薄青 #E8F0FE（囲みの背景） / 濃灰 #202124（文字） / 灰 #5F6368（補助文字）。

### 構成の型（内容に合わせて選ぶ）
- 表紙: タイトル+一言のサブタイトル（日付・発表者はサブタイトルに小さく）。表紙に図形や箇条書きを足さない。
- 章の区切り: 3枚以上の本文が続くまとまりごとに section_header=True のスライドを挟む。
- 箇条書き: 1枚に最大5項目・各項目は短く（目安25字以内）。あふれるなら2枚に分ける。
- 比較（AとB / 前後 / 長所短所）: 箇条書きにせず、左右2つの角丸四角(rounded_rectangle)を並べる。
  左 left=1.7 / 右 left=17.5、width=14.6、薄青 #E8F0FE 塗り+濃灰文字。見出し行だけ青 #1A73E8 塗り+白文字。
- 数値の強調（KPI・実績）: 大きな数字(40pt)+下に小さなラベル(14pt)のtext図形を横に等間隔で並べる（最大4つ）。
- 手順・流れ: 角丸四角を横一列に等間隔で並べ、間を arrow_right でつなぐ（最大4ステップ。超えるなら2段にせず2枚に分ける）。
- データ: 3行以上の数値は本文に書かず ppt_add_table か ppt_add_chart にする。推移=line / 比較=bar / 内訳=pie。
- 文字だけのスライドが3枚続いたら、次は表・グラフ・図解のどれかを入れる。

### 配置のルール
- 端から1.7cm以上の余白を保つ（本文領域: left=1.7〜32.2、タイトル下 top=3.9〜17.3）。
- 座標を勘で決めず、直前に置いた図形の top+height+0.6cm 下に次を積む（重なり防止）。
- 図形を横に並べるときは (使える幅 - 図形幅×個数) を等分して間隔を決める。
- 表・グラフは1枚に1つ。タイトル以外の文と同居させるなら幅を半分にして左右に分ける。

### 仕上げの確認（画像を見られるモデルのみ）
- 全スライド完成後、render_page で表紙+本文2〜3枚を確認し、次があれば直す:
  文字のはみ出し・図形の重なり / 1枚への詰め込みすぎ（→2枚に分割） /
  パレット外の色 / 極端な余白の偏り。直したら再度 render_page で確認する。
